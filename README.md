# 💬 Connect Chat

A modern full-stack chat application built using React, Node.js, Express, and MongoDB.

## 🚀 Features

- User Login
- Send Messages
- Real-time Refresh (Polling)
- MongoDB Atlas Database
- Responsive UI
- Logout
- Message Timestamp
- Modern Chat Interface

## 🛠 Tech Stack

Frontend
- React
- Axios
- CSS

Backend
- Node.js
- Express

Database
- MongoDB Atlas

## Installation

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```
